# Edunova Guardia Mobile

Aplicación móvil base para el rol guardia del sistema Edunova.

## Stack

- React Native
- Expo
- TypeScript
- React Navigation
- Atomic Design

## Estructura

```
src/
  components/
    atoms/
    molecules/
    organisms/
    templates/
  screens/
  navigation/
  services/
  constants/
  utils/
  data/
  types/
```

## Ejecutar

```bash
npm install
npx expo start
```
